Задание №1

В данном задании необходимо написать на ассемблере программу (словарь). Ей на вход дается набор ключей-значений, разделенных пробелами:

key1 value1
key2 value2
key3 value3

Затем после перевода строки, на каждой строке вводится ключ, для которого программа должна вывести соответствующее ему значение. Например:

cat chat
dog chien
mouse souris

dog
chien
cat
chat

Если в качестве ключа передана пустая строка, программа завершается.

В зависимости от варианта, в качестве структур данных для словаря следует использовать либо хеш-таблицу, либо отсортированный массив, по которому делается бинарный поиск.

Вариант Описание
1 Хеш-таблица
2 Быстрая сортировка + бинарный поиск
3 Heapsort + бинарный поиск
4 Сортировка слиянием + бинарный поиск

Номер своего вариант можно вычислить по формуле (N - 1) % 4 + 1, где N — ваш номер в списке вашей группы на странице курса.

В архиве asm-task-1.zip лежит ассемблер (fasm), текстовый редактор (fasmd), отладчик (td), а также пример программы делающей сложение длинных чисел. Пример можно использовать чтобы посмотреть как сделать ввод/вывод.

По желанию можно использовать любой другой ассемблер (nasm, masm, yasm, gas). Также можно писать не 16-битный код, а 32- или 64-битный.
